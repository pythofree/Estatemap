# EstateMap

Mobile application for exploring real estate listings on an interactive map.

## Project goal
Build a commercial-level React Native application with:
- map integration
- property filtering
- user authentication
- backend integration
- offline mode
- push notifications

## Planned tech stack
- React Native + Expo
- Redux Toolkit
- Supabase (Auth + Database)
- Map API (Google Maps / MapView)
- External API (NBP currency rates or OpenWeather)
- AsyncStorage (offline cache)
- SecureStore (token storage)
- Expo Notifications
- Expo Location

## Current status
Project initialization and architecture planning.
